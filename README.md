# Login & Registration System

A beginner-friendly C++ project that allows users to register and log in using file handling.

## Features

- User Registration
- User Login
- Checks for duplicate usernames
- Stores user credentials in a text file
- Verifies username and password
- Menu-driven program

## Technologies Used

- C++
- File Handling
- Functions
- Loops
- Conditional Statements

## Files

- login.cpp
- users.txt (created automatically after registration)

## How to Run

1. Compile the program

```bash
g++ login.cpp -o login
```

2. Run the program

```bash
./login
```
